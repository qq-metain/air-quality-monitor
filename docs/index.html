Air Quality Monitor

A Flutter mobile app that helps users check local air quality, understand pollutant data, review trends, and receive practical AI health advice before going outside.

View GitHub Repository View Screenshots

Problem Statement

Many users check the weather before going outside, but normal weather forecasts often do not clearly explain local air quality. This app helps users decide whether it is suitable to go outside, exercise outdoors, commute, or open windows based on current environmental conditions.

Connected Environment Concept

Mobile Sensing

The app uses the phone's location capability to detect the user's current position.

Environmental Data

Real-time AQI and pollutant information are fetched from an external air quality API.

AI Advice

The app converts raw pollutant values into practical health advice using an AI service.

Data Over Time

History records and weekly trend charts help users understand changes in air quality.

Main Features

Real-time AQI

Shows the current air quality index with a clear status label.

Pollutant Details

Displays PM2.5, PM10, Ozone, NO₂, SO₂, and CO values.

Weekly Trend

Visualises air quality changes over time using a trend chart.

History Records

Saves previous air quality readings locally for later review.

Bilingual Interface

Supports English and Chinese interface switching.

Practical Advice

Gives suggestions about outdoor activity, exercise, and ventilation.

User Journey

The user wants to go outside but is unsure about the air quality. They open the app, log in, tap Get Air Quality, and receive current AQI, pollutant details, and AI health advice. They can then check weekly trends and history records to understand how the environment changes over time.

Technology Stack

Flutter & Dart

Used to build the cross-platform mobile application.

Geolocator

Used for location-based air quality detection.

Open-Meteo Air Quality API

Used to fetch real-time environmental data.

SQLite

Used to store local air quality history records.

DeepSeek API

Used to generate AI health advice from pollutant values.

GitHub Pages

Used to publish this landing page.

Future Improvements

Push Notifications

Warn users when local air quality becomes poor.

Map View

Show nearby air quality conditions visually on a map.

Cloud Sync

Allow users to access their history across multiple devices.

Secure Backend Proxy

Move AI API requests to a backend service for stronger key protection.
